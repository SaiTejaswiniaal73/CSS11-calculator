HTML-CSS-JAVASCRIPT

Tim Berners-Lee

The first version of HTML was written by Tim Berners-Lee in 1993. Since then, there have been many different versions of HTML. The most widely used version throughout the 2000's was HTML 4.01, which became an official standard in December 1999. Another version, XHTML, was a rewrite of HTML as an XML language. XML is a standard markup language that is used to create other markup languages. Hundreds of XML languages are in use today, including GML (Geography Markup Language), MathML, MusicML, and RSS (Really Simple Syndication).

Hakon Wium Lie

SS, or Cascading Style Sheets, first emerged in the early 1990s as a solution to the growing complexity of web design. The web was in its infancy, and designers were limited in their ability to control the appearance of web pages. Web content was mostly text-based, and HTML (HyperText Markup Language) was the primary tool for structuring web documents. However, as the demand for more visually appealing websites grew, the need for a dedicated styling language became apparent. This gave birth to the concept of CSS. In 1994, Hakon Wium Lie, a Norwegian web pioneer, and Bert Bos, a Dutch programmer, proposed the first CSS specification. Their vision was to separate content (HTML) from presentation (CSS), thus simplifying web development and making it more flexible. The difference between HTML and CSS lies in their roles: HTML structures web content, while CSS styles and formats that content.

Brendan Eich

Eich created JavaScript to make web pages more interactive by extending the capabilities of HTML. The language was originally called Mocha, but was later renamed LiveScript and then JavaScript. The name change was a marketing tactic to make JavaScript seem like a supporting language to Java. JavaScript became the ECMA-262 standard in 1997, and the Mozilla Foundation continued to develop it for Firefox. The language's core features and syntax are defined by ECMAScript, a standardized specification that ensures consistency across browsers
